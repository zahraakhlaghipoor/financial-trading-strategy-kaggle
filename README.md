# AI-Powered Financial Trading Strategy
https://drive.google.com/file/d/1tbnePMdXXD5MMCbyVEOycLhCIxdKY5Ki/view?usp=sharing
## Overview

This project presents a data-driven trading strategy that combines Machine Learning and Technical Analysis to identify high-probability trading opportunities in financial markets.

Using historical market data from 1990 to 2024, we developed and evaluated predictive models capable of generating buy and sell signals while incorporating robust risk management techniques. The objective was to improve decision-making, enhance profitability, and reduce trading risk through systematic analysis rather than discretionary trading.

The project was implemented in Python using Jupyter Notebooks and includes extensive data preprocessing, feature engineering, model development, backtesting, and performance evaluation.

---

## Objectives

* Analyze long-term historical financial market data.
* Develop predictive machine learning models for market direction forecasting.
* Generate systematic trading signals.
* Evaluate strategy performance through multi-decade backtesting.
* Implement risk management techniques to improve capital preservation.
* Compare model-driven trading decisions against traditional technical approaches.

## Technical Indicators

The strategy incorporates several widely used technical indicators:

* Simple Moving Average (SMA)
* Exponential Moving Average (EMA)
* Relative Strength Index (RSI)
* Bollinger Bands

These indicators were used as input features to help identify market momentum, trend direction, volatility, and potential reversal zones.

---

## Risk Management Framework

A core focus of this project was risk control and capital preservation.

Implemented techniques include:

* Stop-Loss Management
* Take-Profit Targets
* Trailing Stop-Loss
* Position Sizing
* Risk-to-Reward Optimization

These mechanisms help reduce drawdowns while maximizing potential returns.

## Methodology

### 1. Data Collection

Historical financial market data covering multiple market cycles between 1990 and 2024 was collected and prepared for analysis.

### 2. Data Preprocessing

* Missing value handling
* Feature engineering
* Data normalization
* Technical indicator generation

### 3. Model Training

Machine learning models were trained using engineered features derived from both price action and technical indicators.

### 4. Signal Generation

The trained models generated buy and sell signals based on predicted market behavior.

### 5. Backtesting

Strategies were evaluated across different market conditions:

* Bull Markets
* Bear Markets
* Sideways Markets

Performance metrics were analyzed to assess robustness and consistency.
